> Opens doc in browser
```bash
$ cargo doc --open
```

> Test doc code
```bash
$ cargo t
```

> Publish code in `crates.io`
```bash
$ cargo publish
```

> Stop a particular version from being downloaded
```bash
$ cargo yank --vers [VERSION]
```

> UNDO stop a particular version from being downloaded
```bash
$ cargo yank --vers [VERSION] --undo
```
